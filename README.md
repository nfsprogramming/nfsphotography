# NFS Photography Portfolio

A premium portfolio website for NFS Photography (ESTD 2024).

## Features
- **Premium Dark Design**: High-contrast, sleek aesthetic suitable for high-end photography.
- **Dynamic Animations**: Enhanced with `framer-motion` for smooth reveals and interactions.
- **Responsiveness**: Fully responsive layout for all devices.
- **Gallery**: Lightbox-enabled gallery showcasing portrait, architecture, and macro works.

## Tech Stack
- **Framework**: Vite + React
- **Styling**: Vanilla CSS (Custom Variables & Design System)
- **Icons**: Lucide React
- **Animation**: Framer Motion

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to `http://localhost:5173`

## Project Structure
- `src/components/`: Contains all UI components (Hero, Navbar, Gallery, etc.)
- `src/index.css`: Global styles and theme variables.
- `public/`: Static assets (logo, gallery images).
